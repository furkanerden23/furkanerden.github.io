Create first grid

.container {
display: grid;
grid-template-columns: 1fr 2fr;  /* width based on columns */
grid-template-rows: 1fr 1fr;     /* height based on rows */
gap: 10px;
}

Easily create same rows and columns

Creates 5 rows and 5 columns

•grid-container {
display: grid;
grid-template-rows: repeat(5, 200px);
grid-template-columns: repeat(5, 200px);
}

You can keep out from responsiveness

Elements are not gonna split around the sides

.container {
	width: 800px;
}
Grid sizing

Fixed size

This is not responsive and these two are the same

•grid-container { 
display: grid; 
grid-template-rows: 100px 200px; 
grid-template-columns: 400px 800px;
}
•grid-container { 
display: grid; 
grid-template: 100px 200px / 400px 800 px;
}

Auto size

This means a bit responsiveness

Second row and second column are extensible

•grid-container {
display: grid;
grid-template-rows: 100px auto;       /* fit content */
grid-template-columns: 200px auto;    /* Try to go 100% */
}

Fractional size

This is responsive

•grid-container {
display: grid;
grid-template-rows: 1fr 2fr; 
grid-template-columns: 1fr 2fr; 
}

MinMax size

Second column is gonna be inside of 400px and 800px wide

Responsive in this range

•grid-container {
display: grid;
grid-template-rows: 200px 400px;
grid-template-columns: 200px minmax(400px,800px);
}

Auto grid

It sizes the given value for extra elements

•grid-container {
display: grid;
grid-template-rows: 200px 200px;
grid-template-columns: 200px 200px;
grid-auto-rows: 300px;
}
Grid Placement

Grid container: An element (probably a div) contains all of the elements in our grid layout

Grid item: Every single item in a grid

Tracks: Rows and columns that we created in a grid. There are row tracks and column tracks

Grid cell: Every single unit in a grid layout

Grid lines: Horizontal and vertical lines that creates a grid

Span

Create the first grid

These are the same

.item {
	grid-column: span 2;
}

.item {
	grid-column-start: span 2;
	grid-column-end: auto;
}
.item {
	grid-column: span 2;   /* take 2 cells vertically */
	grid-row: span 2;      /* take 2 cells horizontally */

	grid-column-start: 2;  /* choose the start and ending columns */
	grid-column-end: 4;    /* side column is 1 */
	grid-column-end: -1;   /* this is the right side */

	grid-column-start: 4;  /* its gonna flip the columns order doesnt matter */
	grid-column-end: 2;
}

Order items

Higher order gets closer to the bottom right side

.item {
	order: 1;
}

Grid area

Define the whole grid at one line


.item {
	grid-area: 1 / 2 / 3 / 4;
}


.item {
	grid-row-start: 1;
	grid-column-start: 2;
	grid-row-end: 3;
	grid-column-end: 4;
}

Grid lets you overlap the items if you define them top of each other
