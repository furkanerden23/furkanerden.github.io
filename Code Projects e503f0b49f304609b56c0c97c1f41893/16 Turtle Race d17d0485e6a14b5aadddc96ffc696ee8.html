16. Turtle Race

Introduction

Welcome to the Turtle Race Game project! In this Python endeavor, we'll delve into programming with Python by crafting an engaging game using turtle graphics. As a participant in this race, you'll wager on the outcome by selecting your preferred color turtle, rooting for it to be the first to cross the finish line. Through this project, you'll grasp fundamental programming concepts such as loops, conditional statements, user input, and event handling while enjoying the process of creating your interactive game. Let's embark on this journey together and see who emerges victorious in the race of the colorful turtles!

The aim of this project
  1. Basic Syntax: Students learn about Python syntax, including import statements, loops (for and while), conditional statements (if), and function/method calls.
  1. Turtle Graphics: The project introduces students to turtle graphics, a beginner-friendly way to learn about graphics programming. They learn how to create and manipulate turtle objects, move them around the screen, and change their appearance.
  1. Randomization: Students learn how to generate random numbers using the random module. They use random numbers to simulate the movement of turtles in the race, adding an element of unpredictability to the game.
  1. User Input: The project demonstrates how to obtain user input using the textinput function from the Screen class. Students learn how to prompt the user for input and use that input to drive the behavior of the program.
  1. Conditional Logic: Through the race loop, students learn about conditional logic. They understand how to check conditions (e.g., whether a turtle has crossed the finish line) and take appropriate actions based on those conditions.
  1. Lists and Loops: The project utilizes lists and loops to manage multiple turtles efficiently. Students learn how to iterate over lists of turtles to perform actions on each one.
  1. Event Handling: Although not explicitly shown in this code, the exitonclick() function demonstrates event handling. Students learn how to respond to user events (in this case, clicking on the screen) to control the flow of the program.

Python code
from turtle import Turtle, Screen
import random

is_race_on = False
screen = Screen()
screen.setup(width=500, height=400)
user_bet = screen.textinput(title="Make your bet", prompt="Which turtle will win the race? Enter a color: ")
colors = ["red", "orange", "yellow", "green", "blue", "purple"]
y_positions = [-70, -40, -10, 20, 50, 80]
all_turtles = []

#Create 6 turtles
for turtle_index in range(0, 6):
    new_turtle = Turtle(shape="turtle")
    new_turtle.penup()
    new_turtle.color(colors[turtle_index])
    new_turtle.goto(x=-230, y=y_positions[turtle_index])
    all_turtles.append(new_turtle)

if user_bet:
    is_race_on = True

while is_race_on:
    for turtle in all_turtles:
        #230 is 250 - half the width of the turtle.
        if turtle.xcor() > 230:
            is_race_on = False
            winning_color = turtle.pencolor()
            if winning_color == user_bet:
                print(f"You've won! The {winning_color} turtle is the winner!")
            else:
                print(f"You've lost! The {winning_color} turtle is the winner!")

        #Make each turtle move a random amount.
        rand_distance = random.randint(0, 10)
        turtle.forward(rand_distance)

screen.exitonclick()
