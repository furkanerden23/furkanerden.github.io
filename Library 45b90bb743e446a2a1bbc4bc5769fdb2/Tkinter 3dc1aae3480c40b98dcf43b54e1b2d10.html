def listbox_used(event):                            # Func for listbox
    print(listbox.get(listbox.curselection()))


listbox = Listbox(height=4)                         # Listbox Window
fruits = ["Apple", "Pear", "Orange", "Banana"]      # List of listbox

for item in fruits:                                 # Add items to listbox
    listbox.insert(fruits.index(item), item)        # Give index and item

Place an element on screen

You have to place an element on the screen to be able to see it. You have to use one of them:

  • pack
  • place
  • grid
my_label.pack(side="left")                          # Relative positioning
my_label.place(x=0, y=0)                            # Absoulte positioning
my_label.grid(column=0, row=1, columnspan=1)        # Grid system positioning

Event Driven

How to call a function during window is open

def say_something(a, b, c):
    print(a, b, c)

timer = window.after(1000, say_something, 3, 5, 8)  # it's gonna call 'say_something' every 1000ms (1 sec)

window.after_cancel(timer)                          # Stops the event

Scrollbar in Canvas

You can use a scrollbar with the following:

  • Canvas
  • Text
  • Listbox
window=Tk()

canvas=Canvas(window,bg='#FFFFFF',width=300,height=300,scrollregion=(0,0,500,1500))

back_button_img = PhotoImage(file = f"img/back_button.png")
back_button = canvas.create_image(
    150, 850,
    image=back_button_img)

vbar=Scrollbar(window,orient=VERTICAL)
vbar.pack(side=RIGHT,fill=Y)
vbar.config(command=canvas.yview)

canvas.config(width=300,height=300)
canvas.config(yscrollcommand=vbar.set)
canvas.pack(side=LEFT,expand=True,fill=BOTH)

window.mainloop()
