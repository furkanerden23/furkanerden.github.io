34. Auto Tinder Swiper

Introduction

This Python script designed to automate the process of swiping right (liking) on profiles in the Tinder dating app. Here's a basic introduction to how it works:

  1. Importing Necessary Libraries: The script starts by importing the required modules from the Selenium library. Selenium is a tool for automating web browsers.
  1. Setting Up Variables: It sets up variables for your Facebook login email, Facebook password, and the path to the Chrome WebDriver. Make sure to replace these placeholders with your actual Facebook credentials and the path to your Chrome WebDriver.
  1. Opening Tinder in a Web Browser: It uses the Chrome WebDriver to open the Tinder website.
  1. Logging in with Facebook: It clicks on the login button, then on the Facebook login option, switches to the Facebook login window, fills in your email and password, and logs you in.
  1. Handling Location, Notifications, and Cookies Popups: After logging in, it handles the location permission, notifications, and cookies popups by clicking on the appropriate buttons.
  1. Automated Swiping: It enters a loop where it repeatedly tries to find and click the like (swipe right) button. If it encounters a pop-up indicating a match, it clicks on it to dismiss it. If there are no matches or pop-ups, it waits for a few seconds before trying again.
  1. Exiting the Script: After swiping right a specified number of times (in this case, 100), it quits the web browser session.

Keep in mind that automating interactions with websites like Tinder may violate its terms of service, and using such scripts could result in your account being banned. Always use automation scripts responsibly and be aware of the risks involved.

Features
  1. Automated Login: The script automates the login process to Tinder using Facebook credentials.
  1. Handling Pop-ups: It handles various pop-ups such as location permission, notifications, and cookies, which are common when using Tinder for the first time.
  1. Automated Swiping: The script repeatedly swipes right (likes) on profiles presented in the Tinder interface. It does this in a loop, allowing for continuous swiping without manual intervention.
  1. Error Handling: It includes error handling mechanisms to deal with potential issues like ElementClickInterceptedException (when an element intercepts the click) and NoSuchElementException (when an expected element is not found).
  1. Customizable Parameters: Although not explicitly implemented in this script, you can easily customize parameters such as the number of swipes or the time intervals between swipes to suit your preferences.
  1. Convenience: Automating the swiping process can save users time and effort, especially if they wish to quickly browse through profiles without manually interacting with the app.
  1. Compatibility: The script is designed to work with the Chrome web browser and can be adapted to work with other browsers supported by Selenium WebDriver.

Python code
auto_tinder_swiper.py
from selenium import webdriver
from selenium.webdriver.common.keys import Keys
from selenium.webdriver.common.by import By
from selenium.common.exceptions import ElementClickInterceptedException, NoSuchElementException
from time import sleep

FB_EMAIL = 'YOUR FACEBOOK LOGIN EMAIL'
FB_PASSWORD = 'YOUR FACEBOOK PASSWORD'

chrome_driver_path = 'YOUR CHROME DRIVER PATH'
driver = webdriver.Chrome(executable_path=chrome_driver_path)

driver.get("http://www.tinder.com")

sleep(2)
login_button = driver.find_element(By.XPATH, '//*[@id="content"]/div/div[1]/div/main/div[1]/div/div/header/div[1]/div[2]/div/button')
login_button.click()

sleep(2)
fb_login = driver.find_element(By.XPATH, '//*[@id="modal-manager"]/div/div/div[1]/div/div[3]/span/div[2]/button')
fb_login.click()

sleep(2)
base_window = driver.window_handles[0]
fb_login_window = driver.window_handles[1]
driver.switch_to.window(fb_login_window)
print(driver.title)

email = driver.find_element(By.XPATH, '//*[@id="email"]')
password = driver.find_element(By.XPATH, '//*[@id="pass"]')

email.send_keys(FB_EMAIL)
password.send_keys(FB_PASSWORD)
password.send_keys(Keys.ENTER)

driver.switch_to.window(base_window)
print(driver.title)

sleep(5)
allow_location_button = driver.find_element(By.XPATH, '//*[@id="modal-manager"]/div/div/div/div/div[3]/button[1]')
allow_location_button.click()
notifications_button = driver.find_element(By.XPATH, '//*[@id="modal-manager"]/div/div/div/div/div[3]/button[2]')
notifications_button.click()
cookies = driver.find_element(By.XPATH, '//*[@id="content"]/div/div[2]/div/div/div[1]/button')
cookies.click()

for n in range(100):
    sleep(1)
    try:
        print("called")
        like_button = driver.find_element(By.XPATH, 
            '//*[@id="content"]/div/div[1]/div/main/div[1]/div/div/div[1]/div/div[2]/div[4]/button')
        like_button.click()
    except ElementClickInterceptedException:
        try:
            match_popup = driver.find_element(By.XPATH, ".itsAMatch a")
            match_popup.click()
        except NoSuchElementException:
            sleep(2)

driver.quit()
Instructions for running the project
  1. Install Python: Ensure that Python is installed on your system. You can download and install Python from the official Python website (https://www.python.org/).
  1. Install Selenium: Install the Selenium library using pip, the Python package installer. Open a command prompt or terminal and run the following command:
    pip install selenium
  1. Download Chrome WebDriver: Download the Chrome WebDriver compatible with your Chrome browser version. Make sure to place the WebDriver executable in a directory accessible from your system PATH.
  1. Edit Script: Open the auto_tinder_swiper.py script in a text editor. Replace the placeholders 'YOUR FACEBOOK LOGIN EMAIL', 'YOUR FACEBOOK PASSWORD', and 'YOUR CHROME DRIVER PATH' with your actual Facebook login email, password, and the path to the Chrome WebDriver executable, respectively.
  1. Run Script: Save the changes to the script and execute it. You can run the script by navigating to its directory in the command prompt or terminal and running the following command:
    python auto_tinder_swiper.py
  1. Observe Execution: The script will launch a Chrome browser window, navigate to the Tinder website, log in using the provided Facebook credentials, and start automatically swiping right on profiles. You can observe the script's execution in the command prompt or terminal.
  1. End Execution: Once the script finishes swiping (or you decide to stop it), you can close the browser window or terminate the script execution by pressing Ctrl + C in the command prompt or terminal.
