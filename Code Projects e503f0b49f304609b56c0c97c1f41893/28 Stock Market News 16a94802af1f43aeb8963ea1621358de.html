28. Stock Market News

Introduction

Welcome to the Automated Stock Market News Notifier project! This Python script serves as a robust tool for fetching real-time stock market data and delivering personalized news updates directly to users via text messages. Let's dive into the core functionalities and the underlying coding principles driving this project.

  1. API Integration:
    The backbone of this project lies in its integration with external APIs. We leverage the AlphaVantage API to fetch historical stock market data, including daily closing prices. This data forms the basis for analyzing price movements over time.
  1. Price Analysis:
    Using the fetched stock data, the script calculates the percentage change in the stock price between yesterday and the day before yesterday. This analysis is pivotal in determining whether a significant price change has occurred, warranting further action.
  1. News Retrieval:
    Upon detecting a notable price change, the script dynamically queries the NewsAPI to fetch relevant news articles pertaining to the specified company. This ensures that users receive timely updates aligned with their investment interests.
  1. Message Formatting:
    To provide users with clear and concise updates, the script meticulously formats the retrieved news articles. Each message contains essential details such as the stock name, percentage change, headline, and a brief description of the article.
  1. Twilio Integration:
    The Twilio API serves as the communication bridge between the script and users' mobile devices. Leveraging Twilio's messaging service, the script effortlessly delivers the formatted news updates as text messages, ensuring seamless and instantaneous dissemination of information.
  1. Automation:
    One of the key strengths of this project is its automation capability. By periodically running the script, users can receive timely news updates without manual intervention, streamlining the process of staying informed about the latest market developments.

The Automated Stock Market News Notifier project exemplifies the power of Python in seamlessly integrating with external APIs and automating complex tasks. Through its efficient data retrieval, analysis, and delivery mechanisms, the script empowers users to make informed investment decisions with ease.

Python code
stock_market_news.py
import requests
from twilio.rest import Client

VIRTUAL_TWILIO_NUMBER = "your virtual twilio number"
VERIFIED_NUMBER = "your own phone number verified with Twilio"

STOCK_NAME = "TSLA"
COMPANY_NAME = "Tesla Inc"

STOCK_ENDPOINT = "https://www.alphavantage.co/query"
NEWS_ENDPOINT = "https://newsapi.org/v2/everything"

STOCK_API_KEY = "YOUR OWN API KEY FROM ALPHAVANTAGE"
NEWS_API_KEY = "YOUR OWN API KEY FROM NEWSAPI"
TWILIO_SID = "YOUR TWILIO ACCOUNT SID"
TWILIO_AUTH_TOKEN = "YOUR TWILIO AUTH TOKEN"


#Get yesterday's closing stock price
stock_params = {
    "function": "TIME_SERIES_DAILY",
    "symbol": STOCK_NAME,
    "apikey": STOCK_API_KEY,
}

response = requests.get(STOCK_ENDPOINT, params=stock_params)
data = response.json()["Time Series (Daily)"]
data_list = [value for (key, value) in data.items()]
yesterday_data = data_list[0]
yesterday_closing_price = yesterday_data["4. close"]
print(yesterday_closing_price)

#Get the day before yesterday's closing stock price
day_before_yesterday_data = data_list[1]
day_before_yesterday_closing_price = day_before_yesterday_data["4. close"]
print(day_before_yesterday_closing_price)

#Find the positive difference between 1 and 2. e.g. 40 - 20 = -20, but the positive difference is 20.
difference = float(yesterday_closing_price) - float(day_before_yesterday_closing_price)
up_down = None
if difference > 0:
    up_down = "🔺"
else:
    up_down = "🔻"

#Work out the percentage difference in price between closing price yesterday and closing price the day before yesterday.
diff_percent = round((difference / float(yesterday_closing_price)) * 100)
print(diff_percent)


#If difference percentage is greater than 5 then print("Get News").
if abs(diff_percent) > 1:
    news_params = {
        "apiKey": NEWS_API_KEY,
        "qInTitle": COMPANY_NAME,
    }

    news_response = requests.get(NEWS_ENDPOINT, params=news_params)
    articles = news_response.json()["articles"]

    #Use Python slice operator to create a list that contains the first 3 articles.
    three_articles = articles[:3]
    print(three_articles)

    #Create a new list of the first 3 article's headline and description using list comprehension.
    formatted_articles = [f"{STOCK_NAME}: {up_down}{diff_percent}%\nHeadline: {article['title']}. \nBrief: {article['description']}" for article in three_articles]
    print(formatted_articles)
    client = Client(TWILIO_SID, TWILIO_AUTH_TOKEN)

    #Send each article as a separate message via Twilio.
    for article in formatted_articles:
        message = client.messages.create(
            body=article,
            from_=VIRTUAL_TWILIO_NUMBER,
            to=VERIFIED_NUMBER
        )
Instructions for running the project
  1. Setup Environment:
    • Ensure you have Python installed on your system (preferably Python 3.x).
    • Install required Python packages using pip:
      pip install requests twilio
  1. Obtain API Keys:
    • Sign up for accounts and obtain API keys from the following services:
      • AlphaVantage (for stock market data)
      • NewsAPI (for news articles)
      • Twilio (for sending SMS messages)
  1. Update Constants:
    • Open the stock_market_news.py file in a text editor.
    • Update the following constants with your API keys and other information:
      • STOCK_API_KEY: Your API key from AlphaVantage.
      • NEWS_API_KEY: Your API key from NewsAPI.
      • TWILIO_SID and TWILIO_AUTH_TOKEN: Your Twilio account SID and authentication token.
      • VIRTUAL_TWILIO_NUMBER: Your Twilio virtual number.
      • VERIFIED_NUMBER: Your verified phone number with Twilio.
      • STOCK_NAME: The stock symbol you want to monitor (e.g., "TSLA" for Tesla Inc).
      • COMPANY_NAME: The name of the company corresponding to the stock symbol.
  1. Run the Script:
    • Open a terminal or command prompt.
    • Navigate to the directory containing the stock_market_news.py file.
    • Run the script using the following command:
      python stock_market_news.py
  1. Periodic Execution (Optional):
    • If you want to receive regular updates, you can schedule the script to run periodically using tools like cron jobs (on Unix-based systems) or Task Scheduler (on Windows).
  1. Monitor Updates:
    • Once the script is running, it will periodically check for significant price changes in the specified stock.
    • If a significant change is detected, relevant news articles will be fetched and sent to your mobile phone via SMS.
