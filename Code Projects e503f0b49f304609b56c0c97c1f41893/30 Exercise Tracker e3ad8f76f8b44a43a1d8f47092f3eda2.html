30. Exercise Tracker

Introduction

This project, exercise_tracker.py, is a Python script designed to help users track their exercise routines and record them in a spreadsheet. Here's a basic introduction:

The script starts by importing necessary libraries including requests for making HTTP requests and datetime for handling date and time information. It also imports os for interacting with the operating system.

Next, there are several constants defined such as GENDER, WEIGHT_KG, HEIGHT_CM, and AGE, which represent user-specific information.

The script requires environment variables YOUR_APP_ID, YOUR_API_KEY, and YOUR_SHEET_ENDPOINT to be set, which are used for authentication and API endpoints.

The user is prompted to input a description of exercises they performed.

The script then constructs a request payload with exercise details including the query, gender, weight, height, and age. It sends a POST request to the Nutritionix API to obtain information about the exercises entered.

The obtained exercise data is parsed and printed.

Next, the script records the exercise data into a spreadsheet. It demonstrates three different methods of authentication: no authentication, basic authentication, and bearer token authentication.

  • For no authentication, it simply sends the request.
  • For basic authentication, it includes the username and password obtained from environment variables.
  • For bearer token authentication, it includes a token obtained from environment variables.

Finally, the script prints the response from the spreadsheet API.

This script provides a basic framework for users to track their exercises and store the data in a spreadsheet using various authentication methods.

Python code
exercise_tracker.py
import requests
from datetime import datetime
import os

GENDER = 'YOUR GENDER'
WEIGHT_KG = 'YOUR WEIGHT'
HEIGHT_CM = 'YOUR HEIGHT'
AGE = 'YOUR AGE'

APP_ID = os.environ["YOUR_APP_ID"]
API_KEY = os.environ["YOUR_API_KEY"]

exercise_endpoint = "https://trackapi.nutritionix.com/v2/natural/exercise"
sheet_endpoint = os.environ["YOUR_SHEET_ENDPOINT"]

exercise_text = input("Tell me which exercises you did: ")

headers = {
    "x-app-id": APP_ID,
    "x-app-key": API_KEY,
}

parameters = {
    "query": exercise_text,
    "gender": GENDER,
    "weight_kg": WEIGHT_KG,
    "height_cm": HEIGHT_CM,
    "age": AGE
}

response = requests.post(exercise_endpoint, json=parameters, headers=headers)
result = response.json()
print(result)

today_date = datetime.now().strftime("%d/%m/%Y")
now_time = datetime.now().strftime("%X")

for exercise in result["exercises"]:
    sheet_inputs = {
        "workout": {
            "date": today_date,
            "time": now_time,
            "exercise": exercise["name"].title(),
            "duration": exercise["duration_min"],
            "calories": exercise["nf_calories"]
        }
    }

    #No Auth
    sheet_response = requests.post(sheet_endpoint, json=sheet_inputs)


    #Basic Auth
    sheet_response = requests.post(
        sheet_endpoint, 
        json=sheet_inputs, 
        auth=(
            os.environ["USERNAME"], 
            os.environ["PASSWORD"],
        )
    )

    #Bearer Token
    bearer_headers = {
    "Authorization": f"Bearer {os.environ['TOKEN']}"
    }
    sheet_response = requests.post(
        sheet_endpoint, 
        json=sheet_inputs, 
        headers=bearer_headers
    )

    print(sheet_response.text)

  1. Execute the script by running:
    python exercise_tracker.py
  1. Follow the prompts to input the exercises you performed.
  1. The script will fetch exercise details from the Nutritionix API and print the results.
  1. Exercise data will be recorded in a spreadsheet using the specified authentication method.
