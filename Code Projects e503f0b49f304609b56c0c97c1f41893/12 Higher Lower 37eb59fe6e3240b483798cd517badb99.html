Higher Lower

Introduction

This game tests your social media knowledge. Can you find the accounts that has higher followers. You gonna be surprised by some of the answers.

The aim of this project

You should have a data file. Store the account informations on that file and add lots of famous accounts to make the project bigger. Turn back to main file and choose 2 random accounts from the data file. Do not show the number of followers on the screen, but do the comparison on the background. If the player finds the account which has higher follower, add them 1 point and go for next question. If they can’t find it, game is over.

Achievements
  • Use functions
  • While/for loops
  • If statements
  • Do comparisons
  • Input function usage
  • Store data in variables
  • Print information with F string

main.py
from game_data import data
import random

def get_random_account():
  """Get data from random account"""
  return random.choice(data)

def format_data(account):
  """Format account into printable format: name, description and country"""
  name = account["name"]
  description = account["description"]
  country = account["country"]
  # print(f'{name}: {account["follower_count"]}')
  return f"{name}, a {description}, from {country}"

def check_answer(guess, a_followers, b_followers):
  """Checks followers against user's guess 
  and returns True if they got it right.
  Or False if they got it wrong.""" 
  if a_followers > b_followers:
    return guess == "a"
  else:
    return guess == "b"


def game():
  score = 0
  game_should_continue = True
  account_a = get_random_account()
  account_b = get_random_account()

  while game_should_continue:
    account_a = account_b
    account_b = get_random_account()

    while account_a == account_b:
      account_b = get_random_account()

    print(f"Compare A: {format_data(account_a)}. \n vs \n Against B: {format_data(account_b)}.")
    
    guess = input("Who has more followers? Type 'A' or 'B': ").lower()
    a_follower_count = account_a["follower_count"]
    b_follower_count = account_b["follower_count"]
    is_correct = check_answer(guess, a_follower_count, b_follower_count)

    if is_correct:
      score += 1
      print(f"You're right! Current score: {score}.")
    else:
      game_should_continue = False
      print(f"Sorry, that's wrong. Final score: {score}")

game()

function getRandomAccount() {
    return data[Math.floor(Math.random() * data.length)];
}

function formatData(account) {
    return `${account.name}, a ${account.description}, from ${account.country}`;
}

function checkAnswer(guess, aFollowers, bFollowers) {
    if (aFollowers > bFollowers) {
        return guess === "a";
    } else {
        return guess === "b";
    }
}

function game() {
    let score = 0;
    let gameShouldContinue = true;
    let accountA = getRandomAccount();
    let accountB = getRandomAccount();

    while (gameShouldContinue) {
        accountA = accountB;
        accountB = getRandomAccount();

        while (accountA === accountB) {
            accountB = getRandomAccount();
        }

        const promptMessage = `Compare A: ${formatData(accountA)}.\n vs.\nAgainst B: ${formatData(accountB)}.`;
        const guess = prompt(promptMessage + "\nWho has more followers? Type 'A' or 'B': ").toLowerCase();
        const aFollowerCount = accountA.follower_count;
        const bFollowerCount = accountB.follower_count;
        const isCorrect = checkAnswer(guess, aFollowerCount, bFollowerCount);

        if (isCorrect) {
            score++;
            alert(`You're right! Current score: ${score}.`);
        } else {
            gameShouldContinue = false;
            alert(`Sorry, that's wrong. Final score: ${score}`);
        }
    }
}

game();
