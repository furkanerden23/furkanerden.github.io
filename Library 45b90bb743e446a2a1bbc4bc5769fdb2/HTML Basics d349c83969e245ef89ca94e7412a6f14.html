Attributes

There are specific attributes and global attributes.

Specific attributes like ‘href’ for anchor tag. You can only use that attribute for this tag.

You can use global attributes for any tag, just like ‘draggable’ attribute.

Attributes: https://developer.mozilla.org/en-US/docs/Web/HTML/Global_attributes

<tag attribute=value>Content</tag>

HyperText

You can give another HTML file path to 'href' to direct people to another page of the website

<a href="https://en.wikipedia.org/wiki/Human">Human</a>                 <!-- click 'Human' and go to link -->
<a href="https://en.wikipedia.org/wiki/Human"><img src="images/galaxy.png" alt="My galaxy"></a> <!-- clickable image -->
<a href="#pricing">Pricing</a>                   <!-- It scrolls to element which has the id of 'pricing' -->

<style>
    a {
        text-decoration: none;                              <!-- some css to get rid of underlined anchor -->
    }

Table

No columns, create table row by row

<table>                                                                 <!-- create a table -->

    <thead>                                                             <!-- table heading row -->
        <tr>                                                            <!-- empty table row -->
            <th>Dates</th>                                              <!-- table header cell -->
            <th>Work</th>           
        </tr>           
    </thead>            

    <tbody>                                                             <!-- table body -->
        <tr>                                                            <!-- table row -->
            <td>2010-2013</td>                                          <!-- table data -->
            <td>Lead Developer at Tempo App</td>                        <!-- second cell for this row -->
        </tr>

        <tr>                                                            <!-- open second row -->
            <td>2010</td>
            <td>Researcher at The Institute of Cognitive Neurosciences</td>
        </tr>
    </tbody>

</table>                                                                <!-- close the table -->
