33. Auto Job Application Linkedin

Introduction

This Python script, auto_job_application_linkedin.py, is designed to automate the process of applying to job listings on LinkedIn. Here's a basic overview of how it works:

  1. The script uses the Selenium library to interact with the LinkedIn website. Selenium is a tool for automating web browsers.
  1. It begins by setting up some constants such as your LinkedIn account email, password, phone number, and the path to your Chrome driver.
  1. It then opens a specific LinkedIn job search page related to marketing intern positions in London, England.
  1. The script locates the "Sign in" button and performs the sign-in process using your provided credentials.
  1. After signing in, it searches for job listings on the page and iterates through each listing.
  1. For each listing, it clicks on the listing to view more details.
  1. It attempts to locate and click the "Apply" button on the job listing.
  1. If the "Apply" button is found, it fills in the phone number field with the provided phone number and proceeds to submit the application.
  1. If the application process involves multiple steps, it handles those steps accordingly.
  1. If no "Apply" button is found for a listing, it skips that listing.
  1. After applying to all eligible listings, the script quits the browser.

This script automates the repetitive task of applying to multiple job listings on LinkedIn, saving time and effort for users. However, it's important to use such automation scripts responsibly and ethically, ensuring compliance with LinkedIn's terms of service and respecting the integrity of the job application process.

Features
  1. Automated Job Search: The script opens a specific LinkedIn job search page and retrieves job listings based on predefined search criteria, in this case, marketing intern positions in London, England.
  1. User Authentication: It automates the sign-in process using provided LinkedIn account credentials (email and password).
  1. Dynamic Interaction with Web Elements: Utilizing Selenium, the script locates and interacts with various web elements such as buttons, input fields, and links on the LinkedIn website.
  1. Iterative Job Application: It iterates through each job listing, clicks on the listing to view details, and applies to eligible listings automatically.
  1. Handling Application Process: The script handles the application process, including filling in the phone number field and submitting applications. It also manages multi-step application processes, skipping complex applications if necessary.
  1. Error Handling: The script includes error handling mechanisms such as catching NoSuchElementException to handle cases where certain elements are not found on the page.
  1. Output and Logging: It provides basic console output, printing messages indicating the status of each job application process, such as whether an application was successful or if it was skipped.
  1. Browser Management: The script manages the web browser lifecycle, opening and closing the browser as needed.
  1. Customization: Users can customize parameters such as their LinkedIn account credentials, phone number, and Chrome driver path to adapt the script to their specific needs.

Python code
auto_job_application_linkedin.py
from selenium import webdriver
from selenium.webdriver.common.keys import Keys
from selenium.webdriver.common.by import By
from selenium.common.exceptions import NoSuchElementException
import time

ACCOUNT_EMAIL = 'YOUR LOGIN EMAIL'
ACCOUNT_PASSWORD = 'YOUR LOGIN PASSWORD'
PHONE = 'YOUR PHONE NUMBER'

chrome_driver_path = 'YOUR CHROME DRIVER PATH'
driver = webdriver.Chrome(chrome_driver_path)
driver.get("https://www.linkedin.com/jobs/search/?f_LF=f_AL&geoId=102257491&keywords=marketing%20intern&location=London%2C%20England%2C%20United%20Kingdom&redirect=false&position=1&pageNum=0")

time.sleep(2)
sign_in_button = driver.find_element(By.LINK_TEXT, "Sign in")
sign_in_button.click()

time.sleep(5)
email_field = driver.find_element(By.ID, "username")
email_field.send_keys(ACCOUNT_EMAIL)
password_field = driver.find_element(By.ID, "password")
password_field.send_keys(ACCOUNT_PASSWORD)
password_field.send_keys(Keys.ENTER)

time.sleep(5)

all_listings = driver.find_elements(By.CSS_SELECTOR, ".job-card-container--clickable")

for listing in all_listings:
    print("called")
    listing.click()
    time.sleep(2)
    try:
        apply_button = driver.find_element(By.CSS_SELECTOR, ".jobs-s-apply button")
        apply_button.click()

        time.sleep(5)
        phone = driver.find_element(By.CLASS_NAME, "fb-single-line-text__input")
        if phone.text == "":
            phone.send_keys(PHONE)
        
        submit_button = driver.find_element(By.CSS_SELECTOR, "footer button")
        if submit_button.get_attribute("data-control-name") == "continue_unify":
            close_button = driver.find_element(By.CLASS_NAME, "artdeco-modal__dismiss")
            close_button.click()
            
            time.sleep(2)
            discard_button = driver.find_elements(By.CLASS_NAME, "artdeco-modal__confirm-dialog-btn")[1]
            discard_button.click()
            print("Complex application, skipped.")
            continue
        else:
            submit_button.click()

        time.sleep(2)
        close_button = driver.find_element(By.CLASS_NAME, "artdeco-modal__dismiss")
        close_button.click()

    except NoSuchElementException:
        print("No application button, skipped.")
        continue

time.sleep(5)
driver.quit()
Instructions for running the project
  1. Ensure you have Python installed on your system. You can download and install Python from the official website: https://www.python.org/downloads/
  1. Install the required Python packages by running the following command in your terminal or command prompt:
pip install selenium
  1. Download the Chrome WebDriver that matches your Chrome browser version from the official ChromeDriver website: https://sites.google.com/a/chromium.org/chromedriver/downloads
  1. Extract the downloaded Chrome WebDriver and note down the path to the chromedriver executable.
  1. Open the auto_job_application_linkedin.py file in a text editor.
  1. Replace the placeholders in the script with your LinkedIn account email, password, phone number, and the path to the chromedriver executable.
  1. Save the changes to the script.
  1. Open your terminal or command prompt and navigate to the directory containing the auto_job_application_linkedin.py file.
  1. Run the script by executing the following command:
python auto_job_application_linkedin.py
  1. The script will launch a Chrome browser window, navigate to the LinkedIn job search page, sign in to your LinkedIn account, and start applying to job listings automatically.
  1. Monitor the terminal/command prompt for status messages indicating the progress of the job application process.
  1. Once the script has applied to all eligible job listings, it will close the browser window automatically.
