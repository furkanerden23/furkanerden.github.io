CSS: Cascading Style Sheets

Create CSS

Inline CSS

Inject CSS directly to HTML code with the 'style' attribute

<body style="background-color: #EAF6F6;">

Internal CSS

You can use CSS codes inside of the 'style' tag in an HTML code

Put that in the <head> element

<style>
    body {
        background-color: #EAF6F6;
    }
</style>

External CSS

Call CSS file from HTML code

<link rel="stylesheet" href="css/style.css">

Element Types

Block Elements

Block elements take up all the width as their height

Block elements don't want any other element in their width line

You can change width

<p> </p>                                                                            <!-- common block elements -->
<h1> </h1>
<div> </div>
<ol> </ol>
<form action=""> </form>
p {
        display: inline;                                                            /* You can turn a block element into an inline element */
    }

Inline Elements

Inline elements only take as much width as they need

You can't change the width

<span> </span>                                                                      <!-- common inline elements -->
<img src="" alt="">
<a href=""> </a>
span {
        display: block;                                     /* you can turn a inline element into an block element */
        
        margin: 50px auto 50px auto;                        /* if an inline element centered, you can use auto margin to keep it in center */

        float: left;                                        /* keep element left side and put next element to right side */
        margin-right: 30px;                                 /* balance the space after float */

        clear: right;                                       /* make the element's right side is empty in it's line */
    }

Inline-Block Elements

It takes other elements in the line if there is enough space

You can can change the width

p {
        width: 100px;
        display: inline-block;
    }

Positioning

Static Positioning

Default HTML flow

Relative Positioning

Relative to its first position

Give an element a space between its left edge and where the left edge used to be

It goes over other elements, it does NOT push other elements.

img {
        position: relative;
        left: 30px;
    }

Absolute Positioning

Relative to its parent (kinda)

Relative to nearest positioned(you should give a value to ‚Äòposition‚Äô manually) ancestor(parent) or top left corner of webpage (if all parents are default on ‚Äòposition‚Äô)

Turns element inline-block

img {
        position: absolute;
        right: 30px;
    }

Z-index

Position elements on top of each other

Higher value goes top on other elements on the screen

Default value for all elements is 0

img {
        z-index: 100;
				z-index: 5;
				z-index: -2;
    }

Fixed Positioning

Relative to top left corner

The element stays the same place when the page is scrolled

img {
        position: fixed;
        top: 0;
    }
Media Query

You can give a feature to an element based on a condition, that helps to make a website responsive

Ex: make an h1 colored red under 900px screen width

@media <type> <feature>
@media (max-width: 900px) {
     h1 {
      color: red;
   }
}

Add more than 1 condition

@media (min-width: 900px) and (max-width: 1000px) {
     h1 {
      color: red;
   }
}
