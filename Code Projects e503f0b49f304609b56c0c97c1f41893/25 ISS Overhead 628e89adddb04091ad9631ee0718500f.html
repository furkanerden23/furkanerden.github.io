25. ISS Overhead

Introduction

The iss_overhead.py script is designed to notify users via email when the International Space Station (ISS) is passing overhead during nighttime at their specified location. It utilizes two primary functions: is_iss_overhead() and is_night(), which determine if the ISS is within a certain proximity of the user's latitude and longitude coordinates, and if it is currently nighttime at their location, respectively.

  • Dependencies: The script requires the requests library for making HTTP requests, datetime for handling date and time operations, smtplib for sending emails, and time for implementing a delay.
  • User Configuration: Users need to provide their email credentials (MY_EMAIL and MY_PASSWORD), latitude (MY_LAT), and longitude (MY_LONG) coordinates in the script.
  • Functionality:
    1. is_iss_overhead(): This function retrieves the current position of the ISS from the Open Notify API. It checks whether the ISS is within a specified range (+5 or -5 degrees) of the user's coordinates.
    1. is_night(): This function determines whether it is currently nighttime at the user's location. It retrieves sunrise and sunset times using the Sunrise-Sunset API and compares them with the current time.
  • Main Loop: The script continuously runs in an infinite loop with a sleep interval of 60 seconds. During each iteration, it checks if both conditions (is_iss_overhead() and is_night()) are met. If so, it sends an email notification using the provided SMTP server address.
  • Notification: The email notification includes a subject line indicating the ISS's presence and a simple message prompting the user to look up at the sky.

Note: Users must replace placeholders such as email credentials and SMTP server address with their own information for the script to function correctly. Additionally, they may need to adjust the latitude and longitude coordinates to match their location accurately.

Python code
iss_overhead.py
import requests
from datetime import datetime
import smtplib
import time

MY_EMAIL = "___YOUR_EMAIL_HERE____"
MY_PASSWORD = "___YOUR_PASSWORD_HERE___"
MY_LAT = 51.507351 # Your latitude
MY_LONG = -0.127758 # Your longitude


def is_iss_overhead():
    response = requests.get(url="http://api.open-notify.org/iss-now.json")
    response.raise_for_status()
    data = response.json()

    iss_latitude = float(data["iss_position"]["latitude"])
    iss_longitude = float(data["iss_position"]["longitude"])

    #Your position is within +5 or -5 degrees of the iss position.
    if MY_LAT-5 <= iss_latitude <= MY_LAT+5 and MY_LONG-5 <= iss_longitude <= MY_LONG+5:
        return True


def is_night():
    parameters = {
        "lat": MY_LAT,
        "lng": MY_LONG,
        "formatted": 0,
    }
    response = requests.get("https://api.sunrise-sunset.org/json", params=parameters)
    response.raise_for_status()
    data = response.json()
    sunrise = int(data["results"]["sunrise"].split("T")[1].split(":")[0])
    sunset = int(data["results"]["sunset"].split("T")[1].split(":")[0])

    time_now = datetime.now().hour

    if time_now >= sunset or time_now <= sunrise:
        return True


while True:
    time.sleep(60)
    if is_iss_overhead() and is_night():
        connection = smtplib.SMTP("__YOUR_SMTP_ADDRESS_HERE___")
        connection.starttls()
        connection.login(MY_EMAIL, MY_PASSWORD)
        connection.sendmail(
            from_addr=MY_EMAIL,
            to_addrs=MY_EMAIL,
            msg="Subject:Look Up👆\n\nThe ISS is above you in the sky."
        )

  1. Configure the Script: Open the iss_overhead.py file in a text editor. Replace the placeholder values with your email address (MY_EMAIL), email password (MY_PASSWORD), latitude (MY_LAT), and longitude (MY_LONG). You can also update the SMTP server address (__YOUR_SMTP_ADDRESS_HERE___) if necessary.
  1. Run the Script: Once you've configured the script with your information, save the changes and run the script. You can do this by executing the following command in your terminal or command prompt:
    python iss_overhead.py
  1. Keep the Script Running: The script will start executing and will continuously monitor the ISS's position and nighttime conditions. It will send you an email notification when both conditions (is_iss_overhead() and is_night()) are met.
  1. Stop the Script: To stop the script, you can press Ctrl + C in the terminal or command prompt where it is running.
