20. Mail Merge

Introduction

This Python project is designed to automate the process of generating personalized invitation letters for a birthday party or similar event. The project consists of three main components:

  1. Input Files:
    • invited_names.txt: Contains a list of names of individuals who are being invited to the event. Each name is listed on a separate line.
    • starting_letter.txt: Serves as a template for the invitation letter. It contains placeholder text [name], which will be replaced with each recipient's name.
  1. Python Script (letter_merge.py):
    • This script reads the list of invited names from invited_names.txt.
    • It reads the contents of the invitation letter template from starting_letter.txt.
    • For each name in the list, it replaces the [name] placeholder in the template with the actual name.
    • It then creates a personalized invitation letter for each individual, saving each letter as a separate text file in the ./Output/ReadyToSend/ directory.
  1. Output:
    • After running the script, personalized invitation letters are generated for each individual listed in invited_names.txt. These letters are saved as separate text files in the ./Output/ReadyToSend/ directory.

Overall, this project streamlines the process of generating personalized invitations for the event, saving time and effort compared to manually creating each letter. It demonstrates the use of file handling, string manipulation, and basic automation in Python, particularly in the context of text processing and file I/O.

Features
  • Automated Letter Generation: The project automates the process of generating personalized invitation letters for each invited guest. This eliminates the need for manual letter creation, saving time and effort.
  • Customization: The project allows for customization of each invitation letter by replacing a placeholder in the template letter with the name of each guest. This ensures that each letter is personalized and tailored to the recipient.
  • Input File Handling: The project reads input data from external text files (invited_names.txt and starting_letter.txt). This makes it easy to update the list of invited guests and the content of the template letter without modifying the code.
  • Output File Creation: The project creates individualized invitation letters for each guest and saves them as separate text files in a designated directory (./Output/ReadyToSend/). This facilitates easy access and distribution of the generated letters.
  • Scalability: The project can handle invitations for any number of guests by dynamically generating letters based on the input list of names. This scalability makes it suitable for events of various sizes.

Python code
letter_merge.py
PLACEHOLDER = "[name]"

with open("./Input/Names/invited_names.txt") as names_file:
    names = names_file.readlines()

with open("./Input/Letters/starting_letter.txt") as letter_file:
    letter_contents = letter_file.read()
    for name in names:
        stripped_name = name.strip()
        new_letter = letter_contents.replace(PLACEHOLDER, stripped_name)
        with open(f"./Output/ReadyToSend/letter_for_{stripped_name}.txt", mode="w") as completed_letter:
            completed_letter.write(new_letter)

Instructions for running the project
  1. Ensure you have Python installed on your system. You can download and install Python from the official website: https://www.python.org/downloads/
  1. Download or clone the project files to your local machine.
  1. Place the letter_merge.py, invited_names.txt, and starting_letter.txt files in the same directory.
  1. Open a terminal or command prompt and navigate to the directory where the project files are located.
  1. Run the letter_merge.py script by executing the following command:
    python letter_merge.py
  1. The script will read the list of invited names from invited_names.txt, merge them with the template letter from starting_letter.txt, and generate personalized invitation letters for each guest.
  1. After the script completes execution, you can find the generated invitation letters in the ./Output/ReadyToSend/ directory.
  1. You can now review the generated letters or proceed with sending them to the respective guests.
