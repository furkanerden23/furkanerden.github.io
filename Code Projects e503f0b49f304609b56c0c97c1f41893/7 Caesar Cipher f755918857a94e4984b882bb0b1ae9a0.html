Caesar Cipher

Introduction

Do you wanna cipher your text with your way? It’s easy to do it with this project. We can shift the letters of the text to make it look like ciphered. But we gonna be able to easily shift it back to reach the original text.

The aim of this project

This project wants you to build a function that is able to behave 2 different ways: decode and encode. Get a text from the user. Loop through all of the letters in the text to shift. But you should do it in a certain amount. Then, ask to do it again or not. If its not, end the main while loop.

Achievements
  • Use function
  • While/for loops
  • If statements
  • Shift the letters of a text
  • Input function usage
  • Store data in variables
  • Print information with F string

Javascript code
const alphabet = 'abcdefghijklmnopqrstuvwxyz'.split('');

function caesar(startText, shiftAmount, cipherDirection) {
  let endText = "";
  if (cipherDirection === "decode") {
    shiftAmount *= -1;
  }
  for (let char of startText) {
    if (alphabet.includes(char)) {
      let position = alphabet.indexOf(char);
      let newPosition = (position + shiftAmount + 26) % 26; // To handle negative shifts
      endText += alphabet[newPosition];
    } else {
      endText += char;
    }
  }
  alert(`Here's the ${cipherDirection}d result: ${endText}`);
}

while (true) {
  let direction = prompt("Type 'encode' to encrypt, type 'decode' to decrypt:");
  let text = prompt("Type your message:").toLowerCase();
  let shift = parseInt(prompt("Type the shift number:"));
  shift = shift % 26;

  caesar(text, shift, direction);

  let restart = prompt("Type 'yes' if you want to go again. Otherwise type 'no'.");
  if (restart === "no") {
    alert("Goodbye");
    break;
  }
}
