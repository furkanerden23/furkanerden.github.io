Hangman

Introduction

Remember the good days when we used to play hangman at school? I think that could be cool to code this game. Let’s do it!

The aim of this project

First of all, you should use different python files together to make everything easier. Then, choose a random word from the words file. You gotta be able to display ‘_’ character for every single letter of the target word. Get a letter guess from the player, compare it with the letters of the target word. If the player out of guess, game over. If they find the word, that’s a win!

Achievements
  • import your own file
  • Choice a random item from list
  • While/for loops
  • If statements
  • Length of text
  • Input function usage
  • Store data in variables
  • Print information with F string
Python code
main.py
import random
from hangman_art import stages, logo
from hangman_words import word_list

game_is_finished = False
lives = len(stages) - 1

chosen_word = random.choice(word_list)
word_length = len(chosen_word)

display = []
for _ in range(word_length):
    display += "_"

while not game_is_finished:
    guess = input("Guess a letter: ").lower()

    if guess in display:
        print(f"You've already guessed {guess}")

    for position in range(word_length):
        letter = chosen_word[position]
        if letter == guess:
            display[position] = letter
    print(f"{' '.join(display)}")

    if guess not in chosen_word:
        print(f"You guessed {guess}, that's not in the word. You lose a life.")
        lives -= 1
        if lives == 0:
            game_is_finished = True
            print(f'{chosen_word} was the word!')
            print("You lose.")
    
    if not "_" in display:
        game_is_finished = True
        print("You win.")

    print(stages[lives])

let gameIsFinished = false;
let lives = stages.length - 1;

const chosenWord = wordList[Math.floor(Math.random() * wordList.length)];
const wordLength = chosenWord.length;

const display = Array(wordLength).fill('_');

while (!gameIsFinished) {
  const guess = prompt('Guess a letter: ').toLowerCase();

  if (display.includes(guess)) {
    alert(`You've already guessed ${guess}`);
  }

  for (let position = 0; position < wordLength; position++) {
    const letter = chosenWord[position];
    if (letter === guess) {
      display[position] = letter;
    }
  }

  alert(display.join(' '));

  if (!chosenWord.includes(guess)) {
    alert(`You guessed ${guess}, that's not in the word. You lose a life.`);
    lives--;
    if (lives === 0) {
      gameIsFinished = true;
      alert(`${chosenWord} was the word!\nYou lose.`);
    }
  }

  if (!display.includes('_')) {
    gameIsFinished = true;
    alert('You win.');
  }

  alert(stages[lives]);
}
