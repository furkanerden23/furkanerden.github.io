26. Quizzler

Introduction

This project is a simple quiz application implemented in Python. It allows users to take a quiz consisting of true/false questions. The questions are fetched from an external API, and the user interacts with the quiz through a graphical user interface (GUI) built using the Tkinter library. The application provides immediate feedback on the correctness of the user's answers and keeps track of the user's score throughout the quiz. It offers an engaging and interactive way for users to test their knowledge on different topics.

The quiz application consists of several components:

  1. Question Data Retrieval (data.py):
    • This part of the project fetches quiz questions from an external API (Open Trivia Database) using the requests module.
    • It retrieves a set number of questions with a specified type (in this case, boolean).
  1. Question Model (question_model.py):
    • Defines the Question class to encapsulate individual questions.
    • Each question object contains the text of the question and its correct answer.
  1. Quiz Brain (quiz_brain.py):
    • Implements the core logic of the quiz.
    • Manages the question list, tracks the current question number, and keeps score.
    • Provides methods for checking answers and determining if there are more questions.
  1. User Interface (ui.py):
    • Uses the Tkinter library to create a graphical user interface (GUI) for the quiz.
    • Displays the quiz question and score.
    • Provides buttons for users to select true or false as their answer.
    • Gives visual feedback on the correctness of the user's answer.

Overall, this project offers an engaging way for users to test their knowledge on various topics through a simple and interactive quiz application.

How it works
  1. When the application starts, it retrieves quiz questions from the Open Trivia Database API.
  1. It then presents the first question to the user along with buttons to select true or false.
  1. After the user selects an answer, the application provides immediate feedback on whether the answer was correct or not.
  1. The application proceeds to the next question, updating the score and displaying the next question text.
  1. This process continues until the user completes all the questions or decides to exit the quiz.
Features
  1. Question Retrieval: Questions are fetched from an external API (Open Trivia Database), providing a dynamic pool of questions for users to answer.
  1. Question Model: Questions are encapsulated using a Question class, allowing for easy management and organization of question data.
  1. Quiz Logic: The QuizBrain class manages the core logic of the quiz, including tracking the current question, checking user answers, and updating the score.
  1. User Interface: The application provides a graphical user interface (GUI) using the Tkinter library, making it easy and intuitive for users to interact with the quiz.
  1. Feedback: Users receive immediate feedback on the correctness of their answers, with the application changing the background color to green for correct answers and red for incorrect ones.
  1. Score Tracking: The application keeps track of the user's score throughout the quiz, displaying it prominently on the interface.
  1. Dynamic Content: The quiz can contain a variable number of questions fetched from the API, providing flexibility and variety for users.
  1. True/False Answer Selection: Users can select their answers by clicking on true/false buttons, simplifying the interaction process.

quiz_brain.py
import html


class QuizBrain:

    def __init__(self, q_list):
        self.question_number = 0
        self.score = 0
        self.question_list = q_list
        self.current_question = None

    def still_has_questions(self):
        return self.question_number < len(self.question_list)

    def next_question(self):
        self.current_question = self.question_list[self.question_number]
        self.question_number += 1
        q_text = html.unescape(self.current_question.text)
        return f"Q.{self.question_number}: {q_text}"

    def check_answer(self, user_answer):
        correct_answer = self.current_question.answer
        if user_answer.lower() == correct_answer.lower():
            self.score += 1
            return True
        else:
            return False

ui.py
from tkinter import *
from quiz_brain import QuizBrain

THEME_COLOR = "#375362"


class QuizInterface:

    def __init__(self, quiz_brain: QuizBrain):
        self.quiz = quiz_brain

        self.window = Tk()
        self.window.title("Quizzler")
        self.window.config(padx=20, pady=20, bg=THEME_COLOR)

        self.score_label = Label(text="Score: 0", fg="white", bg=THEME_COLOR)
        self.score_label.grid(row=0, column=1)

        self.canvas = Canvas(width=300, height=250, bg="white")
        self.question_text = self.canvas.create_text(
            150,
            125,
            width=280,
            text="Some Question Text",
            fill=THEME_COLOR,
            font=("Arial", 20, "italic")
        )
        self.canvas.grid(row=1, column=0, columnspan=2, pady=50)

        true_image = PhotoImage(file="images/true.png")
        self.true_button = Button(image=true_image, highlightthickness=0, command=self.true_pressed)
        self.true_button.grid(row=2, column=0)

        false_image = PhotoImage(file="images/false.png")
        self.false_button = Button(image=false_image, highlightthickness=0, command=self.false_pressed)
        self.false_button.grid(row=2, column=1)

        self.get_next_question()

        self.window.mainloop()

    def get_next_question(self):
        self.canvas.config(bg="white")
        if self.quiz.still_has_questions():
            self.score_label.config(text=f"Score: {self.quiz.score}")
            q_text = self.quiz.next_question()
            self.canvas.itemconfig(self.question_text, text=q_text)
        else:
            self.canvas.itemconfig(self.question_text, text="You've reached the end of the quiz.")
            self.true_button.config(state="disabled")
            self.false_button.config(state="disabled")

    def true_pressed(self):
        self.give_feedback(self.quiz.check_answer("True"))

    def false_pressed(self):
        is_right = self.quiz.check_answer("False")
        self.give_feedback(is_right)

    def give_feedback(self, is_right):
        if is_right:
            self.canvas.config(bg="green")
        else:
            self.canvas.config(bg="red")
        self.window.after(1000, self.get_next_question)

Instructions for running the game
  1. Ensure you have Python installed on your system. You can download and install Python from the official Python website (https://www.python.org/downloads/).
  1. Clone or download the project files to your local machine.
  1. Open a terminal or command prompt and navigate to the directory where the project files are located.
  1. Install the required dependencies if they are not already installed. You can install dependencies by running:
    pip install requests
  1. Run the ui.py file using Python:
    python ui.py
  1. The quiz application window should open, allowing you to interact with the quiz. You can answer true/false questions by clicking the corresponding buttons and see immediate feedback on the correctness of your answers.
  1. Once you've completed the quiz, the application will display your final score and indicate that you've reached the end of the quiz.
  1. You can close the application window to exit the quiz.
