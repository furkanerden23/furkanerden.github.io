21. US States Game

Introduction

The U.S. States Game is a simple interactive Python program designed to help users learn the names and locations of the 50 states of the United States. The game presents a blank map of the U.S. with all state names removed. The player is prompted to enter the names of the states one by one. As the player guesses the names correctly, the program reveals the names on the map. The game continues until the player guesses all 50 states or chooses to exit.

Features
  • Interactive User Interface: The game provides a user-friendly interface using the Turtle graphics library, allowing players to interact with the map.
  • Data Handling: The program utilizes the pandas library to read data from a CSV file containing the names and coordinates of the 50 states.
  • State Guessing: Players can guess the names of the states by typing them into a prompt. The program validates the input and updates the map accordingly.
  • State Tracking: The program keeps track of the states guessed correctly by the player.
  • Learning Tool: In addition to playing the game, players can use it as a learning tool to memorize the names and locations of the U.S. states.

Python code
us_states_game.py
import turtle
import pandas

screen = turtle.Screen()
screen.title("U.S. States Game")
image = "blank_states_img.gif"
screen.addshape(image)
turtle.shape(image)

data = pandas.read_csv("50_states.csv")
all_states = data.state.to_list()
guessed_states = []

while len(guessed_states) < 50:
    answer_state = screen.textinput(title=f"{len(guessed_states)}/50 States Correct",
                                    prompt="What's another state's name?").title()
    if answer_state == "Exit":
        missing_states = []
        for state in all_states:
            if state not in guessed_states:
                missing_states.append(state)
        new_data = pandas.DataFrame(missing_states)
        new_data.to_csv("states_to_learn.csv")
        break
    if answer_state in all_states:
        guessed_states.append(answer_state)
        t = turtle.Turtle()
        t.hideturtle()
        t.penup()
        state_data = data[data.state == answer_state]
        t.goto(int(state_data.x), int(state_data.y))
        t.write(answer_state)

Instructions for running the game
  1. Ensure you have Python installed on your system. You can download Python from the official website: https://www.python.org/downloads/
  1. Download or clone the project files onto your computer.
  1. Open a terminal or command prompt and navigate to the directory where the project files are located.
  1. Make sure you have the necessary dependencies installed. You can install the required libraries by running:
    pip install pandas
  1. Run the Python script us_states_game.py by executing the following command:
    python us_states_game.py
  1. The game window should open, displaying a blank map of the United States with the prompt asking you to enter the name of a state. You can start guessing state names by typing them into the prompt.
  1. Continue playing the game by guessing state names until you either guess all 50 states or choose to exit the game by typing "Exit" into the prompt.
  1. If you choose to exit the game, the program will generate a CSV file named states_to_learn.csv, which contains the names of the states you have not yet guessed. This file serves as a learning aid for studying the remaining states.
  1. You can review your progress, modify the code, or run the game again as desired.
  1. Have fun learning and playing the U.S. States Game!
