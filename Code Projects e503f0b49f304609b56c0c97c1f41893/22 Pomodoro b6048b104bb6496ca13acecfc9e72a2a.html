22. Pomodoro

Introduction

The Pomodoro Technique is a time management method developed by Francesco Cirillo in the late 1980s. It uses a timer to break down work into intervals, traditionally 25 minutes in length, separated by short breaks. This project implements a Pomodoro Timer using Python's Tkinter library for the user interface.

  1. Press the "Start" button to begin the Pomodoro Timer.
  1. The timer will switch between work sessions and breaks automatically.
  1. Press the "Reset" button to stop the timer and reset the session count.

This Pomodoro Timer can help users improve productivity by breaking work into manageable intervals and providing regular breaks for relaxation and refocusing.

Features
  • Timer Mechanism: The timer alternates between work sessions and short breaks. After completing a set number of work sessions, it provides a longer break.
  • Countdown Mechanism: Displays the time remaining in the current work or break session.
  • Reset Functionality: Allows the user to reset the timer.
  • Visual Feedback: Provides visual feedback in the form of checkmarks for completed work sessions.
  • User Interface: Utilizes a graphical user interface (GUI) built with Tkinter for easy interaction.
The aim of this project
  1. Understand GUI Development: By using Tkinter, learners can gain experience in creating graphical user interfaces, understanding the layout management, and implementing various widgets like buttons, labels, and canvas.
  1. Practice Event Handling: The project involves handling events such as button clicks to start and reset the timer. Learners can understand how to bind events to specific functions and control the flow of the program based on user interactions.
  1. Implement Time-based Functions: Learners can practice working with time-related functionalities in Python, such as converting minutes to seconds, updating the timer display, and scheduling tasks using the after method.
  1. Explore Conditional Logic: The project includes conditional statements to determine whether the timer should switch to a work session, short break, or long break based on the number of repetitions. Learners can understand and apply conditional logic in Python programming.
  1. Experience Project Structure: By working on a structured project like a Pomodoro Timer, learners can understand how to organize code into functions, manage global variables, and maintain code readability and maintainability.
  1. Apply Time Management Concepts: The project introduces learners to the Pomodoro Technique, a popular time management method. They can explore the concept of breaking work into focused intervals separated by short breaks, which can be applied to enhance personal productivity.

from difflib import context_diff
from tkinter import *
import time
import math
from turtle import color
# ---------------------------- CONSTANTS ------------------------------- #
PINK = "#e2979c"
RED = "#e7305b"
GREEN = "#9bdeac"
YELLOW = "#f7f5dd"
FONT_NAME = "Courier"
WORK_MIN = 25
SHORT_BREAK_MIN = 5
LONG_BREAK_MIN = 20
reps = 0
timer = None

# ---------------------------- TIMER RESET ------------------------------- # 

def reset_timer():
    global reps
    window.after_cancel(timer)
    canvas.itemconfig(timer_text, text="00:00")
    timer_text_label.config(text="Timer", fg=GREEN)
    check_text_label.config(text="")
    reps = 0

# ---------------------------- TIMER MECHANISM ------------------------------- # 

def start_timer():
    global reps
    reps += 1
    
    work_sec = WORK_MIN * 60
    short_break_sec = SHORT_BREAK_MIN * 60
    long_break_sec = LONG_BREAK_MIN * 60
    
    if reps % 8 == 0:
        count_down(long_break_sec)
        timer_text_label.config(text="Break", fg=RED)
    elif reps % 2 == 0:
        count_down(short_break_sec)
        timer_text_label.config(text="Break", fg=PINK)
    else:
        count_down(work_sec)
        timer_text_label.config(text="Work", fg=GREEN)

# ---------------------------- COUNTDOWN MECHANISM ------------------------------- # 

def count_down(count):
    count_min = math.floor(count / 60)
    count_sec = count % 60
    if count_sec < 10:
        count_sec = f"0{count_sec}"
    
    canvas.itemconfig(timer_text, text=f"{count_min}:{count_sec}")
    if count > 0:
        global timer
        timer = window.after(1000, count_down, count - 1)
    else:
        start_timer()
        marks = ""
        work_sessions = math.floor(reps / 2)

# ---------------------------- UI SETUP ------------------------------- #

# Window
window = Tk()
window.title("Pomodoro")
window.config(padx=100, pady=50, bg=YELLOW)

# Canvas
canvas = Canvas(width=200, height=224, bg=YELLOW, highlightthickness=0)

check_text_label.grid(column=1, row=3)

# Button
start_button= Button(text="Start", highlightthickness=0, command=start_timer)
start_button.grid(column=0, row=2)

start_button= Button(text="Reset", highlightthickness=0, command=reset_timer)

Instructions for running the project
  1. Ensure you have Python installed on your system. You can download Python from the official website: https://www.python.org/downloads/
  1. Make sure you have the necessary libraries installed. This project requires the tkinter library, which is usually included with Python installations.
  1. Download the Python script file (pomodoro.py) and the tomato image file (tomato.png) provided in the project.
  1. Place both the Python script file and the tomato image file in the same directory.
  1. Open a terminal or command prompt and navigate to the directory where the Python script file is located.
  1. Run the Python script by executing the following command:
    python pomodoro.py
  1. Once the script is executed, the Pomodoro window should open.
  1. Click the "Start" button to begin the timer. The timer will alternate between work sessions and breaks automatically.
  1. Click the "Reset" button to stop the timer and reset the session count if needed.
  1. You can customize the work session duration (WORK_MIN), short break duration (SHORT_BREAK_MIN), and long break duration (LONG_BREAK_MIN) directly in the Python script if desired.
  1. Enjoy using the Pomodoro to enhance your productivity!
