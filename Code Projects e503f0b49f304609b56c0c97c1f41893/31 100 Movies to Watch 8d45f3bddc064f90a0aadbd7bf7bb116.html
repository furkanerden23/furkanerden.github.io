31. 100 Movies to Watch

Introduction

Web scraping project designed to fetch a list of movie titles from a webpage and save them into a text file named "movies.txt". Here's a breakdown of what the script does:

  1. It imports necessary libraries: requests for making HTTP requests and BeautifulSoup from bs4 for parsing HTML.
  1. It specifies the URL of the webpage containing the list of movies.
  1. The script sends an HTTP GET request to the specified URL and retrieves the HTML content of the webpage.
  1. It uses BeautifulSoup to parse the HTML content.
  1. The script then searches for all HTML elements with the tag <h3> and the class "title", which typically contain movie titles.
  1. It extracts the text (movie titles) from these elements and stores them in a list called movie_titles.
  1. The movies list is created by reversing the order of movie_titles using slicing ([::-1]). This likely reflects the reverse order of the movies on the webpage.
  1. Finally, the script opens a file named "movies.txt" in write mode and writes each movie title from the movies list into the file, with each title on a new line.

In summary, this script scrapes movie titles from a specific webpage and saves them to a text file for further use or analysis.

  1. HTML Parsing: It uses BeautifulSoup to parse the HTML content retrieved from the webpage. This allows the script to navigate through the HTML structure and extract relevant information, such as movie titles.
  1. Data Extraction: The script searches for HTML elements with specific tags and classes (<h3> elements with the class "title") to extract movie titles from the webpage.
  1. Data Manipulation: After extracting movie titles, the script manipulates the data by reversing the order of the titles (likely to match the order as presented on the webpage) using list slicing ([::-1]).
  1. File Handling: It opens a text file named "movies.txt" in write mode and writes each movie title into the file, with each title on a new line.
  1. Reproducibility: The script is designed to be reusable and can be run multiple times to fetch the latest list of movies from the specified URL.
  1. Customization: The script can be easily modified to scrape data from different webpages by changing the URL or adjusting the HTML parsing logic to match the structure of the desired webpage.

Python code
100_movies_to_watch.py
import requests
from bs4 import BeautifulSoup

URL = "https://web.archive.org/web/20200518073855/https://www.empireonline.com/movies/features/best-movies-2/"

# Write your code below this line ðŸ‘‡

response = requests.get(URL)
website_html = response.text

soup = BeautifulSoup(website_html, "html.parser")

all_movies = soup.find_all(name="h3", class_="title")

movie_titles = [movie.getText() for movie in all_movies]
movies = movie_titles[::-1]

with open("movies.txt", mode="w") as file:
    for movie in movies:
        file.write(f"{movie}\n")

  1. Copy the provided Python script (100_movies_to_watch.py) to your local machine.
  1. Open a command line interface (CLI) or terminal, navigate to the directory where the script is located, and execute the script by running the following command:
    python 100_movies_to_watch.py
  1. The script will send a request to the specified URL, scrape the movie titles from the webpage, and save them to a text file named "movies.txt" in the same directory as the script.
  1. Once the script completes execution, you can open the "movies.txt" file to view the list of scraped movie titles.
