36. Instagram Follower

Introduction

This Python script designed to automate the process of following users on Instagram based on the followers of a specified account. Let's break down its functionality:

  1. Imports and Setup:
    • The script imports necessary modules from the Selenium library for web automation in Python.
    • It defines constants such as CHROME_DRIVER_PATH (path to the Chrome WebDriver), SIMILAR_ACCOUNT (the Instagram account whose followers will be targeted), USERNAME, and PASSWORD (your Instagram credentials).
  1. Class Definition - InstaFollower:
    • This class encapsulates methods for logging in to Instagram, finding followers of a specified account, and following those followers.
    • The constructor initializes a Chrome WebDriver instance using the provided driver path.
  1. Login Functionality:
    • The login() method navigates to the Instagram login page, enters the provided username and password, and submits the form to log in.
  1. Finding Followers:
    • The find_followers() method navigates to the profile of the specified account (SIMILAR_ACCOUNT), locates the followers button, and clicks on it to open the followers modal.
    • It then scrolls through the modal to load more followers. Currently, it scrolls 10 times, which can be adjusted as needed.
  1. Following Users:
    • The follow() method finds all follow buttons on the page and attempts to click them one by one.
    • If an "ElementClickInterceptedException" occurs, it likely means that the button is not clickable, possibly due to a pop-up. In such cases, it attempts to close the pop-up by finding and clicking the cancel button.
  1. Execution:
    • Finally, an instance of InstaFollower is created with the Chrome WebDriver path provided.
    • The login(), find_followers(), and follow() methods are called sequentially to execute the automation process.

Note:

  • Before running the script, you need to provide the path to your Chrome WebDriver (CHROME_DRIVER_PATH), your Instagram username (USERNAME), and your Instagram password (PASSWORD).
  • Also, ensure that you've installed the necessary packages (selenium) and have the Chrome WebDriver installed and configured correctly.
  • Automating interactions with Instagram may violate its terms of service, and aggressive automation could lead to your account being banned or restricted. Use such scripts with caution and responsibility.

  1. Follower Discovery: It navigates to the profile of a specified account and retrieves the list of followers by clicking on the followers button and scrolling through the modal to load more followers.
  1. Following Automation: The script automates the process of following users by clicking on the follow buttons of the retrieved followers.
  1. Error Handling: It includes error handling for potential issues such as ElementClickInterceptedException, which may occur when trying to click follow buttons. The script attempts to handle such exceptions by clicking on the cancel button if a pop-up interferes with the follow action.

  1. Scalability: The script can be easily modified to adjust the number of times it scrolls through the followers modal to load more followers or to add additional functionalities such as unfollowing users or liking posts.
  1. Automation Efficiency: By automating the process of following users based on a specified account's followers, the script saves time and effort compared to manual interaction, especially for accounts with a large number of followers.

Python code
instagram_follower.py
from selenium import webdriver
from selenium.webdriver.common.keys import Keys
from selenium.webdriver.common.by import By
from selenium.common.exceptions import ElementClickInterceptedException
import time

CHROME_DRIVER_PATH = 'YOUR CHROM DRIVER PATH'
SIMILAR_ACCOUNT = "buzzfeedtasty"
USERNAME = 'YOUR INSTAGRAM USERNAME'
PASSWORD = 'YOUR INSTAGRAM PASSWORD'


class InstaFollower:

    def __init__(self, path):
        self.driver = webdriver.Chrome(executable_path=path)

    def login(self):
        self.driver.get("https://www.instagram.com/accounts/login/")
        time.sleep(5)

        username = self.driver.find_element(By.NAME, "username")
        password = self.driver.find_element(By.NAME, "password")

        username.send_keys(USERNAME)
        password.send_keys(PASSWORD)

        time.sleep(2)
        password.send_keys(Keys.ENTER)

    def find_followers(self):
        time.sleep(5)
        self.driver.get(f"https://www.instagram.com/{SIMILAR_ACCOUNT}")

        time.sleep(2)
        followers = self.driver.find_element(By.XPATH, '//*[@id="react-root"]/section/main/div/header/section/ul/li[2]/a')
        followers.click()

        time.sleep(2)
        modal = self.driver.find_element(By.XPATH, '/html/body/div[4]/div/div/div[2]')
        for i in range(10):
            self.driver.execute_script("arguments[0].scrollTop = arguments[0].scrollHeight", modal)
            time.sleep(2)

    def follow(self):
        all_buttons = self.driver.find_elements(By.CSS_SELECTOR, "li button")
        for button in all_buttons:
            try:
                button.click()
                time.sleep(1)
            except ElementClickInterceptedException:
                cancel_button = self.driver.find_element(By.XPATH, '/html/body/div[5]/div/div/div/div[3]/button[2]')
                cancel_button.click()


bot = InstaFollower(CHROME_DRIVER_PATH)
bot.login()
bot.find_followers()
bot.follow()
