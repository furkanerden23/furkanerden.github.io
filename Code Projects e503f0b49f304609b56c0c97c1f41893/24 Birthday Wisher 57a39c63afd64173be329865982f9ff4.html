24. Birthday Wisher

Introduction

The "Birthday Wisher" project is a Python script designed to automate the process of sending birthday wishes to individuals via email. It utilizes data stored in a CSV file to identify birthdays and personalized letter templates to craft heartfelt messages. The script leverages Python's libraries for handling dates, reading CSV files, and sending emails via SMTP to streamline the birthday wishing process.

Functionality:

  1. Data Retrieval: The script reads birthday data from a CSV file named "birthdays.csv". This file contains information such as the name, email address, and date of birth (month and day) for individuals.
  1. Birthday Detection: Using the current date, the script compares it with the birthday data from the CSV file to identify if any birthdays match the current date.
  1. Template Selection: Once a birthday match is found, the script randomly selects one of the available letter templates (e.g., "letter_1.txt", "letter_2.txt", etc.) from the "letter_templates" directory.
  1. Personalization: The selected letter template contains placeholders (e.g., "[NAME]") which are replaced with the name of the birthday person retrieved from the CSV file, creating a personalized message for each recipient.
  1. Email Sending: After personalizing the message, the script establishes a connection to an SMTP server (using the email sender's credentials provided in the script) and sends the birthday email to the recipient's email address. The email subject is set to "Happy Birthday!" and includes the personalized message generated from the selected letter template.

Purpose:
The purpose of the "Birthday Wisher" project is to automate the task of sending birthday wishes to friends, family members, or colleagues. By automating this process, the script ensures that birthday greetings are sent promptly and consistently, eliminating the need for manual intervention and reducing the risk of forgetting to send wishes to loved ones. Additionally, the use of personalized letter templates adds a thoughtful touch to the birthday messages, enhancing the overall experience for the recipients.

Features
  1. Automated Birthday Detection: The script automatically detects if there are any birthdays on the current date by comparing it with the birthday data stored in the CSV file.
  1. Customizable Letter Templates: Users can create and customize letter templates according to their preferences. The script randomly selects one of these templates to generate a personalized birthday message for each recipient.
  1. Personalized Messages: Each birthday email is personalized with the recipient's name, retrieved from the CSV file, ensuring that the wishes feel genuine and heartfelt.
  1. Dynamic Email Content: The content of the birthday emails dynamically changes based on the selected letter template and the name of the recipient, creating a unique message for each person.
  1. SMTP Email Sending: The script utilizes SMTP (Simple Mail Transfer Protocol) to send birthday emails directly from the user's email account. This feature ensures that the emails are sent securely and reliably.
  1. Easy Configuration: Users can easily configure the script by providing their email address, password, and SMTP server address. These details enable the script to send emails on behalf of the user.
  1. CSV Data Management: The project allows users to manage birthday data efficiently using a CSV file. Users can add, update, or remove birthday entries as needed, simplifying the management of birthday information.
  1. Scalability: The script can handle a large number of birthday entries, making it suitable for personal or professional use where there are multiple birthdays to track and celebrate.

Python code
birthday_wisher.py
from datetime import datetime
import pandas
import random
import smtplib

MY_EMAIL = "YOUR EMAIL"
MY_PASSWORD = "YOUR PASSWORD"

today = datetime.now()
today_tuple = (today.month, today.day)

data = pandas.read_csv("birthdays.csv")
birthdays_dict = {(data_row["month"], data_row["day"]): data_row for (index, data_row) in data.iterrows()}
if today_tuple in birthdays_dict:
    birthday_person = birthdays_dict[today_tuple]
    file_path = f"letter_templates/letter_{random.randint(1,3)}.txt"
    with open(file_path) as letter_file:
        contents = letter_file.read()
        contents = contents.replace("[NAME]", birthday_person["name"])

    with smtplib.SMTP("YOUR EMAIL PROVIDER SMTP SERVER ADDRESS") as connection:
        connection.starttls()
        connection.login(MY_EMAIL, MY_PASSWORD)
        connection.sendmail(
            from_addr=MY_EMAIL,
            to_addrs=birthday_person["email"],
            msg=f"Subject:Happy Birthday!\n\n{contents}"
        )

Instructions for running the project
  1. Set Up Environment:
    • Ensure you have Python installed on your system. You can download it from the official Python website if you haven't already.
  1. Install Required Libraries:
    • Make sure you have the required libraries installed. This project relies on the pandas library for handling CSV files and the smtplib library for sending emails. You can install these libraries using pip:
      pip install pandas
  1. Prepare Data and Templates:
    • Create a CSV file named "birthdays.csv" containing the birthday data in the format specified in the script (name, email, year, month, day).
    • Create letter templates (e.g., "letter_1.txt", "letter_2.txt", etc.) in the "letter_templates" directory. Ensure each template contains placeholders like "[NAME]" for personalized messages.
  1. Update Script:
    • Open the "birthday_wisher.py" script in a text editor.
    • Replace the placeholders in the script:
      • Replace "YOUR EMAIL" with your email address.
      • Replace "YOUR PASSWORD" with your email password. (Note: Storing passwords directly in scripts is not recommended for security reasons. Use environment variables or a more secure method in production.)
      • Replace "YOUR EMAIL PROVIDER SMTP SERVER ADDRESS" with the SMTP server address provided by your email service provider.
  1. Run the Script:
    • Open a terminal or command prompt.
    • Navigate to the directory containing the "birthday_wisher.py" script.
    • Run the script using the following command:
      python birthday_wisher.py
  1. Follow Prompts:
    • If today is someone's birthday listed in the CSV file, the script will select a random letter template, personalize the message, and send an email to the birthday person's email address.
    • Follow any prompts or messages displayed in the terminal/command prompt during script execution.
  1. Review Sent Emails:
    • Check your email account to ensure that the birthday emails were sent successfully and that they contain the expected personalized messages.
  1. Automation:
    • For automation, you can schedule the script to run daily using tools like cron (on Unix-based systems) or Task Scheduler (on Windows). This ensures that birthday wishes are sent automatically without manual intervention.
