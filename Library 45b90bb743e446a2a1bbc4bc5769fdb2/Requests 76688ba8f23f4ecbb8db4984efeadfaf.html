Basics

import requests

response = requests.get(url="http://api.open-notify.org/iss-now.json")      # Request data from link

Responses

  • 1XX: Hold on
  • 2XX: No Error
  • 3XX: Go away
  • 4XX: You screwed up, user mistake
  • 5XX: I screwed up, a website mistake
response.status_code                                        # Returns one of the responses above
response.raise_for_status()                                 # Show error if there is an error

Data of request

data = response.json()                                      # How to get request as json(dictionary)
